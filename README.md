# Geopoto

Offline-first geography training app built with Expo, pnpm workspaces, Turbo,
oxlint, oxfmt, and tsgo.

## Workspaces

- `apps/mobile`: Expo React Native app shell
- `packages/geo-data`: Static geography data package shell
- `tooling/tsconfig`: Shared TypeScript configs
- `tooling/oxlint`: Shared lint package placeholder

## Commands

```sh
pnpm install
pnpm check
```
